# Wrangler Homebrew Tap

```bash
brew install --cask wrangler-control/tap/wrangler
```

That's it. The app keeps itself up to date ("Restart & update" in-app), so
`brew upgrade` won't need to touch it.

Wrangler is the control tower for engineers running parallel coding agents —
see [wrangler-control.com](https://wrangler-control.com) or grab the DMG
directly from [Releases](https://github.com/wrangler-control/releases/releases/latest).
